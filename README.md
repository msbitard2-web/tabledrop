# Полиция Дропа

Веб-приложение для учёта дропа по локациям (6 часов на локацию).

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере адрес, который покажет Vite (обычно http://localhost:5173).

## Страницы

- **Локации** — список локаций, создание новой. В каждой локации: 6 почасовых таблиц (название дропа, изображение, количество, цена за шт, сумма), итог за час, общая таблица за 6 часов и общий итог. Отмечаются 1, 2 и 3 место по сумме за час.
- **Рейтинги** — вкладки: самая прибыльная локация, самый большой минимальный дроп, самый большой максимальный дроп, самое большое количество серебра. Призёры выделены.
- **Медали** — список медалей (Нубо-лока, Локация на миллион, на 2/5 млн, ИМБА-ЛОКА, 1кк/2кк/3кк за час, HAPPY HOUR). Условия выдачи можно описать позже.

Данные хранятся в `localStorage` в браузере.
